A set of CLI tools for testing the MDMulti platform.

## Installation

`npm i -g mdmcli`

Or alternatively...

`npm i -g mdmulti/cli` (bleeding from master, will be buggy)

## Usage

`mdmcli --help`

## Cloning

- `git clone --recurse-submodules https://github.com/mdmulti/cli`
